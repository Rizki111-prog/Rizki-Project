# **App Name**: FinTrack Pro

## Core Features:

- Sales Processing: Main interface to handle new sales transactions.
- Expense Logging: Log and manage operational expenses.
- Financial Dashboard: Financial reports dashboard showing profit/loss and balance.
- Transaction History: Log page to view, filter, and track past activities.
- Responsive Sidebar Navigation: Collapsible sidebar for navigation, toggleable on mobile.
- Active State Indicator: Visually highlight the currently active menu item based on the URL.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: Teal (#009688) to highlight key interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif font known for its modern, neutral appearance and excellent readability.
- Use Lucide-react icons for menu items in the sidebar.
- Persistent sidebar layout using Next.js layout.tsx to avoid re-rendering.
- Subtle transition animations for sidebar collapse and expansion.
